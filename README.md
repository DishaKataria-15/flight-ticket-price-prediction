# ✈️ Flight Ticket Price Prediction using Machine Learning

A Machine Learning project that analyzes and predicts flight ticket prices using various flight-related factors and regression techniques.

---

# 🚀 Project Overview

This project focuses on analyzing different factors affecting flight ticket prices and building a machine learning model to predict flight fares.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Engineering
- Feature Scaling
- Regression Modeling
- Model Evaluation

---

# 🛠️ Technologies Used

- Python

- Pandas

- Seaborn
- Scikit-learn
- Statsmodels
- Jupyter Notebook

# ⚙️ Machine Learning Workflow

The project follows a complete ML workflow:

1. Data Cleaning
2. Feature Engineering
3. Label Encoding
4. Feature Scaling
5. Multicollinearity Check using VIF
6. Train-Test Split
7. Linear Regression Model
8. Model Evaluation

---

# 📊 Evaluation Metrics

The following metrics were used:

- R2 Score
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)

---

# 🔍 Key Insight

Flight ticket prices are affected by:
- Airline
- Travel Class
- Source City
- Destination City
- Days Left Before Journey

The Linear Regression model was able to explain a significant portion of flight price variation.
